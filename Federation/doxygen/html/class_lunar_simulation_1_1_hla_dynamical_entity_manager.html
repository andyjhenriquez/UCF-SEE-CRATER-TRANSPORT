virtual LIBAPI void LunarSimulation::HlaDynamicalEntityManager::addHlaDynamicalEntityDefaultInstanceListener ( HlaDynamicalEntityListenerPtr  listener)
pure virtual

Add a listener that will automatically be added to all instances of HlaDynamicalEntity (Both local and remote).

The listener is notified when any attribute of an instance of HlaDynamicalEntity is updated. The listener is also called when an interaction is sent to an instance of HlaDynamicalEntity. All listeners will automatically be removed when an instance is removed. This method is idempotent.

Parameters
listenerThe listener to add.

pure virtual

Add a valueListener that will automatically be added to all instances of HlaDynamicalEntity (Both local and remote).

The Valuelistener is notified when any attribute of an instance of HlaDynamicalEntity is updated. Note that the attribute values that are notified are not guaranteed to be the latest available values for that attribute. All listeners will automatically be removed when an instance is removed. This method is idempotent.

◆ isEnabled()

virtual LIBAPI bool LunarSimulation::HlaDynamicalEntityManager::isEnabled ( )
pure virtual

Check if the HlaDynamicalEntity manager is actually enabled when connected.

An HlaDynamicalEntity manager may be disabled with the setEnabled(false) function, or it may be configured as if available in the FOM by the setEnabledIfAvailableInFom() function and then not available in the FOM at runtime.

This method should only be used when connected, it will always return false when not connected.

Returns
true if connected and the HlaDynamicalEntity manager is enabled.
