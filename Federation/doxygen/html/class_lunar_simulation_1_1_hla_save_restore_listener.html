virtual LIBAPI void LunarSimulation::HlaSaveRestoreListener::federationNotRestored ( HlaRestoreFailure  restoreFailureReason,
bool  localRestoreWasSuccessful,
HlaLogicalTimePtr  localRestoredTime,
bool  wasAdvancing 
)
pure virtual

Federation was not restored, one or more federates could not restore its state.

The federates that failed to restore its state are now in an undefined state, this may affect the entire federation.

If the federate was not in the advancing time state and the save/restore was time managed, the federate might need to wait until the granted time state is reached.

Parameters
restoreFailureReasonreason for the failure
localRestoreWasSuccessfultrue if the local federate restore was successful
localRestoredTimelogical time when the local federate was restored
wasAdvancingtrue if the local federate was in the advancing time state when the restore was performed
