◆ addHlaMoonListener()

virtual LIBAPI void LunarSimulation::HlaMoon::addHlaMoonListener ( HlaMoonListenerPtr  listener)
pure virtual

Adds a listener.

The listener is notified when any attribute of this object is updated. This listener is also called when an interaction is sent to this object. This method is idempotent.

To make sure that no attribute updates arrive before adding the listener, use: HlaMoonManager#addHlaMoonDefaultInstanceListener(HlaMoonListenerPtr listener).

Parameters
listenerThe listener to add.

◆ addHlaMoonValueListener()

virtual LIBAPI void LunarSimulation::HlaMoon::addHlaMoonValueListener ( HlaMoonValueListenerPtr  valueListener)
pure virtual

Adds a valueListener.

The valueListener is notified when any attribute of this object is updated. Note that the attribute values that are notified are not guaranteed to be the latest available values for that attribute. This method is idempotent.

To make sure that no attribute updates arrive before adding the listener, use: HlaMoonManager#addHlaMoonDefaultInstanceValueListener(HlaMoonValueListenerPtr valueListener).

Parameters
valueListenerThe valueListener to add.
