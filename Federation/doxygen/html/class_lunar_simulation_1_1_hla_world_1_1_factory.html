Creates a HlaWorld with the specified settings and timestamps.

The parameter manualEvokeMode decides if the instance will run in manual or auto evoke mode.

Auto evoke mode

All notifications for this HlaWorld instance will be delivered in the same thread, a thread that is created by HlaWorld. The notifications will be delivered automatically , as fast as possible. No need to call any evoke methods.

Manual evoke mode

In this mode notifications will be delivered when one of the evoke methods are called. The notification is delivered in the same thread that makes the call to the notification method. Note that the HlaWorld will be updated asynchronously, without need for evoke.

If HlaSettings has been overridden with a FederateConfig.txt file in the current directory, the supplied settings will not be used. A new settings object will be created using the settings specified in the file.

Exceptions thrown in the user notification code will be posted to the ExceptionListener.
