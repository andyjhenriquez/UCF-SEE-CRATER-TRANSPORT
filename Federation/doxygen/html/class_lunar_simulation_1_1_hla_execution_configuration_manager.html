◆ addHlaExecutionConfigurationDefaultInstanceListener()

virtual LIBAPI void LunarSimulation::HlaExecutionConfigurationManager::addHlaExecutionConfigurationDefaultInstanceListener ( HlaExecutionConfigurationListenerPtr  listener)
pure virtual

Add a listener that will automatically be added to all instances of HlaExecutionConfiguration (Both local and remote).

The listener is notified when any attribute of an instance of HlaExecutionConfiguration is updated. The listener is also called when an interaction is sent to an instance of HlaExecutionConfiguration. All listeners will automatically be removed when an instance is removed. This method is idempotent.

Parameters
listenerThe listener to add.

Add a valueListener that will automatically be added to all instances of HlaExecutionConfiguration (Both local and remote).

The Valuelistener is notified when any attribute of an instance of HlaExecutionConfiguration is updated. Note that the attribute values that are notified are not guaranteed to be the latest available values for that attribute. All listeners will automatically be removed when an instance is removed. This method is idempotent.

◆ isEnabled()

virtual LIBAPI bool LunarSimulation::HlaExecutionConfigurationManager::isEnabled ( )
pure virtual

Check if the HlaExecutionConfiguration manager is actually enabled when connected.

An HlaExecutionConfiguration manager may be disabled with the setEnabled(false) function, or it may be configured as if available in the FOM by the setEnabledIfAvailableInFom() function and then not available in the FOM at runtime.

This method should only be used when connected, it will always return false when not connected.

Returns
true if connected and the HlaExecutionConfiguration manager is enabled.

Enables or disables the HlaExecutionConfiguration manager.

The manager is enabled by default. The state can only be changed when not connected. When the HlaExecutionConfiguration manager is disabled it will act as disconnected, ie throws HlaNotConnectedException.
