◆ addHlaLauncherDefaultInstanceListener()

virtual LIBAPI void LunarSimulation::HlaLauncherManager::addHlaLauncherDefaultInstanceListener ( HlaLauncherListenerPtr  listener)
pure virtual

Add a listener that will automatically be added to all instances of HlaLauncher (Both local and remote).

The listener is notified when any attribute of an instance of HlaLauncher is updated. The listener is also called when an interaction is sent to an instance of HlaLauncher. All listeners will automatically be removed when an instance is removed. This method is idempotent.

Parameters
listenerThe listener to add.

Add a valueListener that will automatically be added to all instances of HlaLauncher (Both local and remote).

The Valuelistener is notified when any attribute of an instance of HlaLauncher is updated. Note that the attribute values that are notified are not guaranteed to be the latest available values for that attribute. All listeners will automatically be removed when an instance is removed. This method is idempotent.

◆ isEnabled()

virtual LIBAPI bool LunarSimulation::HlaLauncherManager::isEnabled ( )
pure virtual

Check if the HlaLauncher manager is actually enabled when connected.

An HlaLauncher manager may be disabled with the setEnabled(false) function, or it may be configured as if available in the FOM by the setEnabledIfAvailableInFom() function and then not available in the FOM at runtime.

This method should only be used when connected, it will always return false when not connected.

Returns
true if connected and the HlaLauncher manager is enabled.
