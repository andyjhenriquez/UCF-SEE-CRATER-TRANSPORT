virtual LIBAPI void LunarSimulation::HlaPayloadManager::addHlaPayloadDefaultInstanceListener ( HlaPayloadListenerPtr  listener)
pure virtual

Add a listener that will automatically be added to all instances of HlaPayload (Both local and remote).

The listener is notified when any attribute of an instance of HlaPayload is updated. The listener is also called when an interaction is sent to an instance of HlaPayload. All listeners will automatically be removed when an instance is removed. This method is idempotent.

Parameters
listenerThe listener to add.

◆ addHlaPayloadDefaultInstanceValueListener()

pure virtual

Add a valueListener that will automatically be added to all instances of HlaPayload (Both local and remote).

The Valuelistener is notified when any attribute of an instance of HlaPayload is updated. Note that the attribute values that are notified are not guaranteed to be the latest available values for that attribute. All listeners will automatically be removed when an instance is removed. This method is idempotent.

◆ isEnabled()

virtual LIBAPI bool LunarSimulation::HlaPayloadManager::isEnabled ( )
pure virtual

Check if the HlaPayload manager is actually enabled when connected.

An HlaPayload manager may be disabled with the setEnabled(false) function, or it may be configured as if available in the FOM by the setEnabledIfAvailableInFom() function and then not available in the FOM at runtime.

This method should only be used when connected, it will always return false when not connected.

Returns
true if connected and the HlaPayload manager is enabled.

virtual LIBAPI void LunarSimulation::HlaPayloadManager::setEnabled ( bool  enabled)
pure virtual

Enables or disables the HlaPayload manager.

The manager is enabled by default. The state can only be changed when not connected. When the HlaPayload manager is disabled it will act as disconnected, ie throws HlaNotConnectedException.

Parameters
enabledtrue to enable the HlaPayload manager, otherwise false
