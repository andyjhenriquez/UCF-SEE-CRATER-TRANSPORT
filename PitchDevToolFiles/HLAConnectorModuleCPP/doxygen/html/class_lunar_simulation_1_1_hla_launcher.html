◆ addHlaLauncherListener()

virtual LIBAPI void LunarSimulation::HlaLauncher::addHlaLauncherListener ( HlaLauncherListenerPtr  listener)
pure virtual

Adds a listener.

The listener is notified when any attribute of this object is updated. This listener is also called when an interaction is sent to this object. This method is idempotent.

To make sure that no attribute updates arrive before adding the listener, use: HlaLauncherManager#addHlaLauncherDefaultInstanceListener(HlaLauncherListenerPtr listener).

Parameters
listenerThe listener to add.

◆ addHlaLauncherValueListener()

virtual LIBAPI void LunarSimulation::HlaLauncher::addHlaLauncherValueListener ( HlaLauncherValueListenerPtr  valueListener)
pure virtual

Adds a valueListener.

The valueListener is notified when any attribute of this object is updated. Note that the attribute values that are notified are not guaranteed to be the latest available values for that attribute. This method is idempotent.

To make sure that no attribute updates arrive before adding the listener, use: HlaLauncherManager#addHlaLauncherDefaultInstanceValueListener(HlaLauncherValueListenerPtr valueListener).

Parameters
valueListenerThe valueListener to add.
